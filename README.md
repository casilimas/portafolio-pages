Frontend Application for MongoDB Integration
Descripción
Esta aplicación frontend está diseñada para interactuar con una base de datos MongoDB, tanto en entornos de desarrollo como de producción. La comunicación 
y las restricciones de datos vienen directamente del backend proporcionado en Back-api. Se puede utilizar MongoDB Compass o MongoDB Atlas 
para gestionar y visualizar los datos.
Requisitos
Node.js
npm (Node Package Manager)
Acceso a MongoDB (Compass o Atlas)
Instalacion:
git clone https://github.com/casilimas/portafolio-pages
Instala las dependencias:
Copiar código
